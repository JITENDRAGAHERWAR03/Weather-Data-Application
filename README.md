# Weather Data Application

A Python-based weather monitoring application that retrieves real-time weather information using the OpenWeatherMap API. The application allows users to view current weather conditions, save weather records to CSV files, and compare temperatures across multiple cities through graphical visualizations.

## Features

* Real-time weather data retrieval
* Temperature, humidity, and weather condition display
* CSV export functionality
* Temperature comparison graphs
* Error handling for API requests
* Simple command-line interface

## Technologies Used

* Python
* OpenWeatherMap API
* Requests
* CSV
* Matplotlib
